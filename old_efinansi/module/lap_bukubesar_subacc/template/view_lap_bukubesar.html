Petunjuk

  • Pada pengisian sub akun, dapat digunakan angka dan huruf apapun. Secara default huruf 'x' yang digunakan
    • Pengisian elemen sub akun berupa angka, akan mencari elemen sub akun dengan angka sesuai inputan
      • Misal 01-00-01-02-02-04
      • Seluruh elemen sub akun akan dicari berdasarkan inputan angka tersebut
    • Pengisian elemen sub akun berupa huruf, akan mencari seluruh data berdasarkan elemen sub akun tersebut
      • Misal xx-01-02-00-02-00
      • Elemen pertama sub akun akan dicari semua datanya, kemudian elemen lainnya karena menggunakan angka, akan dicari berdasarkan inputan angka tersebut
    • Pengisian elemen sub akun berupa huruf dan angka, akan mencari seluruh data berdasarkan elemen sub akun tersebut ( dianggap sama dengan pengisian elemen sub akun menggunakan huruf seperti contoh di atas )
      • Misal 1x-2a-02-00-02-00
      • Elemen pertama dan kedua sub akun akan dicari semua datanya, kemudian elemen lainnya karena menggunakan angka, akan dicari berdasarkan inputan angka tersebut
